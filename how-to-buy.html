    How to Buy Mink Coin

    Step-by-Step Instructions

    Step 1: Set Up a Crypto Wallet

    If you don't already have a crypto wallet, you'll need to set one up. We recommend using MetaMask:

    Go to the MetaMask Website and download the wallet extension for your browser.

    Follow the instructions to create a new wallet and secure your recovery phrase.

    (Remember to never share your recovery phrase with anyone!)

    Step 2: Add Ethereum Classic Network

    You now need to add the Ethereum Classic network to MetaMask:

    Open MetaMask and click on the network dropdown at the top left hand corner.

    Select + Add network then in the settings page click Add a network manually and enter the following details:

    Network Name: Ethereum Classic

    New RPC URL: https://www.ethercluster.com/etc

    Chain ID: 61

    Symbol: ETC

    Block Explorer URL: https://blockscout.com/etc/mainnet

    Click Save

    Step 3: Buy Ethereum Classic (ETC)

    Purchase Ethereum Classic (ETC) from a cryptocurrency exchange such as Binance, Kraken, Bitget, or Coinbase. Once you have ETC, send it to your MetaMask wallet address.

    Step 4: Swap ETC for Mink Coin

    Use HebeSwap to swap ETC for Mink Coin:

    Go to Hebe Swap

    Connect your MetaMask wallet.

    Enter the amount of ETC you want to swap and complete the transaction.

    Import Mink Coin into MetaMask token address is: 0x89E50A895d0FE5e5A278B2935225f50c063Ef372
